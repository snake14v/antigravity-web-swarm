### Week 3-4: Deployment Wave 1

**Goal:** Install 2 more ShopSense units (paid customers)

**Per-Deployment Process:**
1. **Pre-visit checklist (2 days before):**
   - Hardware tested (camera focus, Pi boot, network)
   - Shop owner briefing (email: what to expect, setup time, training time)
   - Take photos of shop (for case study)

2. **Install day (4 hours on-site):**
   - 30 min hardware setup
   - 30 min network config (WiFi or hotspot)
   - 30 min camera positioning (for best detection)
   - 30 min dashboard training (show owner how to use OLOG)

3. **Post-install (7 days):**
   - Daily check-in via WhatsApp
   - Remote troubleshooting if any issues
   - Collect 3 photos of deployment
   - Get shop owner testimonial (quote or short video)

**Output:**
- 3 ShopSense units deployed (1 pilot + 2 paid)
- 3 case studies drafted
- 3 shop owner testimonials collected

**Success Metric (Week 4):**
- [ ] 2 paid customers (₹50K revenue)
- [ ] 100% installation success
- [ ] Shop owners report positive first week

## Contingency Plans

### If Sales Are Slow (only 1 customer by June)
- Pivot to different customer segment (restaurants? pharmacies?)
- Drop price to ₹15K to accelerate adoption
- Focus on open source (GitHub stars = credibility for next round)

### If Product Has Issues in Field
- Pause sales, fix bugs (1-2 weeks)
- Get feedback from deployed units (feature requests)
- Release patch version (v1.0.2)
- Resume sales after stabilization

### If Can't Find Hardware Tech to Hire
- Do deployments yourself (slow, but maintain control)
- Partner with local tech shop for installations (revenue share)
- Consider outsourcing support via Freshdesk

### If Burnout Hits
- Hire freelancer for social media (₹15K/month)
- Focus on sales (revenue-generating) over content
- Set 6-hour/day work limit to prevent burnout
