# 🧑‍💼 Hiring Plan — Ooru Logix

## Philosophy
Hire only when AI agents can't handle the workload. Hire for physical presence, client trust, and domain expertise that AI lacks.

---

## Revenue Triggers

| Phase | Revenue Trigger | Hires | Monthly Cost |
|-------|----------------|-------|-------------|
| 1 - Solo + AI | ₹0 → ₹1L/month | None | ₹0 |
| 2 - First Boots | ₹1L → ₹5L/month | 2 | ~₹40K |
| 3 - Growth | ₹5L → ₹20L/month | 2 more | ~₹1.2L |
| 4 - Scale | ₹20L+ | 2 more | ~₹3L |

---

## Phase 2 Hires (Revenue > ₹1L/month)

### 1. Field Engineer / Technician
- **Role:** Install ShopSense at client sites, maintain hardware
- **Skills:** Basic electronics, Raspberry Pi, networking, customer service
- **Salary:** ₹15,000–20,000/month
- **Why AI can't do this:** Physical installation requires hands

### 2. Sales Associate / BD
- **Role:** Walk into chai shops, demo the system, close deals
- **Skills:** Kannada fluency, local market knowledge, persuasion
- **Salary:** ₹15,000–20,000/month + commission
- **Why AI can't do this:** Trust requires face-to-face in this market

---

## Phase 3 Hires (Revenue > ₹5L/month)

### 3. Junior Developer
- **Role:** Frontend (React/mobile), dashboard features, API integrations
- **Skills:** React, TypeScript, Python, Git
- **Salary:** ₹30,000–40,000/month
- **Why now:** Feature velocity needs exceed AI agent capacity

### 4. Marketing Intern
- **Role:** Social media execution, content scheduling, Canva/CapCut editing
- **Skills:** Instagram/LinkedIn savvy, basic design, video editing
- **Salary:** ₹10,000–15,000/month
- **Why now:** Consistent posting schedule requires daily execution

---

## Phase 4 Hires (Revenue > ₹20L/month)

### 5. ML Engineer
- **Role:** Model training, accuracy improvement, new detection categories
- **Skills:** PyTorch, YOLOv8, OpenCV, data annotation
- **Salary:** ₹50,000–80,000/month

### 6. Operations Manager
- **Role:** Client success, support tickets, deployment coordination
- **Skills:** Project management, customer service, Bangalore knowledge
- **Salary:** ₹30,000–50,000/month
