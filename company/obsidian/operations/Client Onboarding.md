# 🤝 Client Onboarding — Ooru Logix

## Onboarding Workflow (ShopSense)

### Stage 1: Discovery (Day 1-3)
1. Initial contact (WhatsApp/Phone/LinkedIn)
2. Understand shop type, size, and current process
3. Quick qualification:
   - Does the shop sell cigarettes, chai, or snacks? (must have at least 2)
   - Is there local WiFi or can we set up a router?
   - Does the owner want to reduce shrinkage / automate billing?
4. Schedule site visit

### Stage 2: Site Audit (Day 4-7)
1. Visit the shop in person
2. Photograph: counter layout, tray positions, lighting conditions
3. Measure: distances, camera mounting options, power outlet locations
4. Note: brand selection, price list, customer flow patterns
5. Assess: WiFi signal strength, electrical capacity
6. Create site audit report

### Stage 3: Proposal (Day 7-10)
1. Send hardware quote (₹15K–25K based on kit tier)
2. Explain: no monthly fees, no internet needed, 6-month warranty
3. Show demo video or live demo at another site
4. Address concerns:
   - "What if the camera breaks?" → Graceful fallback to manual
   - "Will it slow down my billing?" → 3-second detection time
   - "Who owns my data?" → You do. 100%. Local storage only.
5. Get verbal confirmation

### Stage 4: Setup (Day 10-14)
1. Order/assemble hardware kit
2. 3D print custom enclosure based on site measurements
3. Pre-configure Pi nodes with shop's price list
4. Install hardware at shop (2-3 hours)
5. Run calibration: 
   - Camera positioning
   - Detection accuracy test (10 items each category)
   - UI brightness/contrast for lighting conditions
6. Train the owner on touchscreen usage
7. Hand over owner's phone dashboard URL

### Stage 5: Support (Day 14+)
1. WhatsApp group for quick support
2. First-week daily check-in
3. Weekly check-in for month 1
4. Monthly check-in after that
5. Remote dashboard monitoring (Phase 3)

## Onboarding for Web Design Clients

### Stage 1: Inquiry
1. Client finds us via website, LinkedIn, or referral
2. Fillout at oorulogix.com/register
3. Initial scope call (15 min)

### Stage 2: Quote
1. Define deliverables (pages, features, timeline)
2. Send quote based on tier (₹499 / ₹4,999 / ₹14,999 / custom)
3. Collect advance (50% for projects > ₹4,999)

### Stage 3: Build
1. Design mockup → client approval
2. Build → internal QA
3. Deploy to staging → client review
4. Iterate → final approval
5. Deploy to production

### Stage 4: Handoff
1. Transfer domain/hosting credentials
2. Provide documentation
3. 30-day post-launch support included
