## Quick Reference

### Brand Voice Cheat Sheet
**DO**:
- "We lost ₹15,000 on our first order because we didn't understand retail."
- "Hari's tea shop in Rajajinagar went from 2-hour inventory counts to 8 minutes."
- "ShopSense works offline. That's it."

**DON'T**:
- "Our innovative edge AI solutions leverage distributed computing paradigms."
- "Optimize your end-of-day reconciliation processes."
- "We're thrilled to announce exponential growth!"

### Forbidden Words
No: disrupt, paradigm shift, leverage, synergy, ecosystem, seamless, cutting-edge, maximize, optimize, revolutionize, solutions

### Required Vocabulary
Yes: edge AI, shop owner, offline-first, real-time, margin, billing, works, simple, Bangalore retail, bootstrap, chai shop, kirana

### Color Palette
- **Navy**: #0B2648 (primary)
- **Chai Brown**: #8B6F47 (secondary)
- **Green**: #2DB76F (accent)
- **White**: #FFFFFF (background)

### Fonts
- **Headings**: Inter SemiBold (600) or Lato Bold (700)
- **Body**: Inter Regular (400) or Lato Regular (400)
- **Code**: JetBrains Mono

## Workflows

### How to Create a LinkedIn Post
1. Open `content-engine/templates/linkedin-templates.md`
2. Choose one of 10 templates
3. Adapt to current situation (use real shop names, real metrics)
4. Draft in Google Doc, tag `#brand-review`
5. Brand Guardian reviews with checklist (2-3 hours SLA)
6. Schedule via LinkedIn native tool
7. Track impressions/engagement daily
8. Respond to all comments within 4 hours

### How to Create an Instagram Post
1. Open `content-engine/templates/instagram-templates.md`
2. Choose format (reel, carousel, testimonial, meme, etc.)
3. Create visual assets in Figma (1080x1350px for feed, 1080x1920px for stories)
4. Write caption using template
5. Draft in Figma / Canva, export optimized images
6. Post to Instagram, add caption + hashtags
7. Respond to comments/DMs within 1 hour
8. Track reach/engagement daily

### How to Write a Blog Post
1. Open `content-engine/templates/blog-outlines.md`
2. Pick a week/topic
3. Follow H2/H3 structure provided
4. Write 2000+ words targeting primary keyword
5. Add 3-5 internal links
6. Create featured image (1200x630px, <150KB)
7. Run through `seo-monitor/SKILL.md` on-page checklist
8. Submit to Brand Guardian for voice/visual review
9. Publish and repurpose into 3 LinkedIn + 5 Instagram assets
10. Track organic traffic daily (target: 200+ users by week 4)

### How to Repurpose a Blog Post
1. Published blog post → Extract 3-5 key insights
2. Create 3 LinkedIn posts (spacing 2-4 days apart)
   - Post 1: Teaser/hook
   - Post 2: Stat/insight
   - Post 3: Thread or deeper dive
3. Design 5 Instagram assets
   - Reel (15-30 sec demo/explainer)
   - Carousel (step-by-step or before/after)
   - Quote graphic (stat or key insight)
   - Story/behind-the-scenes
   - Testimonial or customer win
4. Draft 2 WhatsApp messages
   - Message 1 (Monday): Initial announcement
   - Message 2 (Friday): Follow-up with case study/result
5. Schedule across 2-4 weeks to avoid fatigue
6. Track performance across channels

### How to Audit Content Before Publishing
1. Open `brand-guardian/SKILL.md`
2. Use **Content Review Checklist**:
   - Brand Voice Check (no jargon, right tone)
   - Visual Check (colors, readability, images)
   - Accuracy Check (facts, metrics, links)
   - CTA Check (one clear call-to-action)
   - Channel-Specific Check (LinkedIn/Instagram/Blog/WhatsApp rules)
3. Score using **Brand Consistency Scoring Rubric** (10-point scale)
4. Return feedback or approve

### How to Track Monthly Performance
1. Open `seo-monitor/SKILL.md` → **Monthly SEO Reporting Template**
2. Pull data:
   - Google Search Console (keyword rankings, clicks, impressions)
   - Google Analytics (organic traffic, bounce rate, conversions)
   - Social media insights (LinkedIn, Instagram, WhatsApp)
   - Backlink tools (Semrush/Ahrefs: new links, domain authority)
3. Fill in template sections:
   - Keyword rankings
   - Organic traffic
   - Pages & content performance
   - Technical SEO status
   - Backlink profile
   - Competitive intelligence
   - Issues & action items
   - Wins & learnings
4. Share in Slack #marketing channel
5. Identify winners, plan next month
